# Heterogeneous Scientific Foundation Model Collaboration

This paper introduces Eywa, a heterogeneous agentic framework designed to enable collaboration between language-centric agents and domain-specific foundation models across scientific domains. Eywa augments non-linguistic foundation models with a language-based reasoning interface, allowing them to participate in high-level inference and decision-making. The system includes EywaAgent, a drop-in replacement for single-agent pipelines, EywaMAS for multi-agent system integration, and EywaOrchestra for dynamic orchestration of heterogeneous experts. Experiments demonstrate that Eywa improves task performance across physical, life, and social sciences while reducing reliance on language-only reasoning, token usage, and execution time. The framework enables native collaboration across different modalities, addressing the challenge of communication barriers in agentic AI systems.
