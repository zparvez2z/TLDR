# MUSE-Autoskill: Self-Evolving Agents via Skill Creation, Memory, Management, and Evaluation

This paper introduces MUSE-Autoskill, a skill-centric LLM agent framework that treats skills as long-lived, testable assets governed by a full lifecycle: creation, memory, management, evaluation, and refinement. The system integrates on-demand skill generation with runtime execution, per-skill memory accumulation, unit-test-driven validation, and adaptive context management to support long-horizon tasks. Experiments on SkillsBench and SkillLearnBench show state-of-the-art performance among GPT-5.5-backed agents, strong gains from both human and self-created skills, superior stability, and effective cross-agent skill transfer. Results demonstrate that lifecycle-managed skills can distill agent experience into reusable capabilities that outperform human-authored skills on covered subsets.
