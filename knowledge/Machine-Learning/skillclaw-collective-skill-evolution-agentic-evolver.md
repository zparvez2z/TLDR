# [2604.08377] SkillClaw: Let Skills Evolve Collectively with Agentic Evolver open search open navigation menu contact arXiv subscribe to arXiv mailings

SkillClaw is a novel framework designed to enable continuous collective evolution of reusable skills in multi-user large language model (LLM) agent ecosystems. Unlike static skill sets, SkillClaw aggregates cross-user interactions over time to identify and propagate skill improvements autonomously. This system allows for dynamic refinement and extension of skills based on recurring behavioral patterns from diverse user experiences, facilitating knowledge transfer and cumulative capability enhancement. Experimental results demonstrate notable performance improvements in real-world agent scenarios using limited interaction data. By centralizing skill updates in a shared repository, SkillClaw ensures system-wide benefit without additional user effort.
