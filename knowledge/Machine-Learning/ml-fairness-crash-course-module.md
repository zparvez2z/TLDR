# Fairness | Machine Learning | Google for Developers

This module explains core principles of machine learning fairness, focusing on how human biases can appear in datasets and models. It teaches practitioners to proactively examine data for bias before training and to evaluate model predictions with fairness metrics. The content emphasizes responsible AI practices and provides strategies to identify and mitigate bias, culminating in hands-on exercises and assessments.
- Audit training data and model predictions for bias prior to deployment.
- Recognize common human biases that ML systems can replicate.
- Identify and apply mitigation strategies to reduce bias in datasets and models.
- Evaluate fairness using metrics such as demographic parity, equality of opportunity, and counterfactual fairness.
- Includes a programming exercise and quiz (~110 minutes total) and assumes familiarity with foundational ML concepts.
