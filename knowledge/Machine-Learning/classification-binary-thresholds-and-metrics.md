# Classification | Machine Learning | Google for Developers

This module teaches how to turn logistic regression probability outputs into binary class predictions, select decision thresholds, and evaluate models with standard classification metrics. It explains the confusion matrix and the trade-offs between precision and recall, and shows how to interpret ROC curves and AUC. The content also introduces prediction bias and provides a brief overview of multi-class classification, building on prior modules on linear and logistic regression. Practical guidance helps you choose metrics and thresholds that align with real-world objectives.
- Convert logistic regression probabilities into binary labels using decision thresholds.
- Use the confusion matrix to analyze errors and trade-offs (false positives vs. false negatives).
- Calculate and interpret accuracy, precision, recall, F1, ROC curves, and AUC.
- Recognize and reason about prediction bias in classification.
- Get a brief introduction to multi-class classification and its relation to binary classification.
- Prerequisites: Intro to ML, Linear Regression, and Logistic Regression.
