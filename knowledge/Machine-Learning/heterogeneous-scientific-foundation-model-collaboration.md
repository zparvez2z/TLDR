# Heterogeneous Scientific Foundation Model Collaboration

This paper introduces Eywa, a novel heterogeneous agentic framework designed to enable collaboration between language-based agents and domain-specific foundation models across scientific domains. By augmenting specialized foundation models with language-based reasoning interfaces, Eywa facilitates higher-level reasoning over diverse data modalities beyond natural language. The framework supports single-agent (EywaAgent), multi-agent (EywaMAS), and dynamic orchestration (EywaOrchestra) setups, improving utility while reducing token consumption and execution time compared to language-only systems. Extensive experiments across physical, life, and social sciences demonstrate significant performance gains and efficient multi-modal collaboration.
- Eywa bridges the gap between language-centric AI systems and specialized scientific foundation models.
- EywaAgent combines language models with specialized predictors for better inference and decision-making.
- EywaMAS enables flexible multi-agent systems with heterogeneous model collaboration.
- EywaOrchestra introduces dynamic planning for orchestrating diverse agents to solve complex tasks.
- Experiments show improved accuracy, reduced token usage (~30%), and lower execution times (~10%) over language-only baselines.
