# Logistic Regression | Machine Learning | Google for Developers

This module introduces logistic regression, a supervised learning approach for predicting the probability of a binary outcome rather than a continuous value. It explains how the sigmoid function maps model outputs to probabilities and why log loss is preferred over squared loss for classification. The content contrasts linear and logistic regression, emphasizes the importance of regularization to prevent overfitting, and outlines common use cases. It assumes prior knowledge of introductory ML and linear regression and is designed to be completed in about 35 minutes.

- Predicts probabilities for binary outcomes (e.g., spam detection, rain prediction)
- Uses the sigmoid function to convert linear outputs (logits) into probabilities
- Optimizes with log loss instead of squared loss for classification tasks
- Highlights the role of regularization to reduce overfitting and improve generalization
- Compares when to use linear vs. logistic regression
- Covers calculating probability, loss and regularization, and knowledge checks
- Prerequisites: Intro to Machine Learning and Linear Regression modules
- Estimated duration: 35 minutes
