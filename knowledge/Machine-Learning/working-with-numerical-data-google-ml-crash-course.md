# Working with numerical data | Machine Learning | Google for Developers

This Google ML Crash Course module explains how to prepare numerical data for machine learning, from ingestion via feature vectors to practical feature engineering. It emphasizes data cleaning and transformation tasks such as normalization, binning, and scrubbing, and introduces creating synthetic features with polynomial transforms. The module clarifies the distinction between numerical and categorical data and outlines the qualities of effective numerical features. A basic understanding of machine learning is recommended before starting.
