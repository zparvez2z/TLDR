# Linear regression | Machine Learning | Google for Developers

This module introduces linear regression as a fundamental technique to model the relationship between input features and a continuous label. It explains the linear model equation, the roles of bias and weights, and how training adjusts these parameters to minimize prediction error. The course covers loss functions, gradient descent optimization, and hyperparameter tuning, with practical examples like predicting car fuel efficiency. It also extends to models with multiple features and includes interactive and programming exercises to reinforce learning.
- Linear regression models predictions with a linear equation (y' = b + Σ wᵢxᵢ), where bias and weights are learned parameters.
- Bias corresponds to the intercept; weights correspond to slopes, and both are optimized during training.
- A loss function measures prediction error, and gradient descent iteratively minimizes this loss to find optimal parameters.
- Hyperparameter tuning (e.g., learning rate) is essential for training efficiency and stability.
- The module demonstrates single- and multi-feature use cases (e.g., car MPG predicted by weight, displacement, acceleration, cylinders, horsepower) and provides interactive and coding exercises.
