# Introduction to Machine Learning | Google for Developers

This foundational course introduces core machine learning (ML) concepts without covering implementation or data handling. It explains the major types of ML and focuses on the fundamentals of supervised learning. The material highlights how ML-based problem solving differs from traditional rule-based programming. Learners can reinforce concepts with a "Test Your Understanding" section.
- Understand different types of ML (e.g., supervised, unsupervised, reinforcement)
- Grasp key supervised learning concepts such as features, labels, training, evaluation, and metrics
- Learn how ML workflows and iterative model improvement differ from traditional programming approaches
- Conceptual focus only; no coding, tooling, or data wrangling is covered
- Part of Google Developers' Machine Learning Foundational courses
